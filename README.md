# MANTO
Create an open-source text-to-video AI generation tool. 

The core purpose of `MANTO` is to be a storytelling tool. It is desinged to create an interactive narrative, animation, novel prompt-mechanism and open-weight model. In a nutshell, `MANTO` serves as an open-source competitor for the Gemini veo 3.
